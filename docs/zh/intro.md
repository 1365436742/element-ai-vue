# 简介

**Element AI Vue** 是一个基于 Vue 3 的 AI 聊天组件库，专为快速构建智能对话界面而设计。无论是企业级应用还是个人项目，Element AI Vue 都能帮助你轻松打造专业的 AI 交互体验。所有内容均开源。没有任何闭源子包。

## ✨ 特性

### 🖥️ 多端适配

- **PC 端**：完美适配桌面端浏览器，提供流畅的交互体验
- **移动端**：响应式设计，自动适配各种移动设备屏幕

### 🌍 多语言支持

- 内置国际化（i18n）支持
- 轻松切换不同语言环境
- 支持自定义语言包扩展

### 🎨 主题定制

- **亮色主题**：清新明亮的视觉风格
- **暗色主题**：护眼舒适的深色模式
- 支持主题切换，自动跟随系统偏好

### 🎯 高度可定制

- 灵活的样式定制能力
- CSS 变量支持，轻松修改主题色彩
- 组件插槽丰富，满足各种自定义需求

### 🧩 丰富的组件

- **Bubble** - 消息气泡组件
- **BubbleList** - 消息列表组件
- **Sender** - 消息发送框组件
- **Markdown** - Markdown 渲染组件
- **CodeHighlight** - 代码高亮组件
- **CodeMermaid** - Mermaid 图表组件
- **ThoughtChain** - 思维链组件
- **Thinking** - 思考状态组件
- **FilesCard** - 文件卡片组件
- **FilesUpload** - 文件上传组件
- **DragUpload** - 拖拽上传组件
- 更多组件持续开发中...

### 🚀 开发体验

- 完整的 TypeScript 支持
- 详尽的文档和示例
- 开箱即用，快速集成

## 🎯 适用场景

- AI 智能客服系统
- ChatGPT 类对话应用
- 企业知识库问答
- 智能助手应用
- 任何需要 AI 对话功能的 Web 应用
